# signalsolo-cn
Inclusive Learning Platform
Designed with care for students with dyspraxia and other learning differences

-----Ideological Concept (The "Why")----
Traditional learning platforms are built for the neurotypical brain, often creating physical and cognitive barriers for students with Dyspraxia and ADHD.
NeuroFlow is a "Cognitive Orthotic" that adapts to the user's motor and executive function needs. Our mission is to move from "speed-based testing" to "depth-based learning" by providing tools that bypass fine-motor struggles and task-initiation paralysis.

-----Technologies Used-----
🎨 Frontend (Inclusive UI/UX)
React (TypeScript) → For a stable, type-safe interface.

Tailwind CSS → Custom high-contrast Navy & Pastel theme for visual stability.

Atkinson Hyperlegible Font → Optimized for character recognition.

Vite → For high-speed performance and low-latency interactions.

----Backend & Architecture-----
Frontend-First Architecture → Minimized server-roundtrips to prevent focus-breaking lags.

Node.js & npm → Dependency management for accessibility libraries.

Git/GitHub → Version control and collaborative deployment.

-----The Six Pillars of NeuroFlow-----
1. AI-Driven Natural Command Assistant
The Feature: A voice and text-based command terminal.

Inclusive Impact: Allows users to "Navigate to Modules" or "Schedule Labs" via natural speech, bypassing complex, small-target menu navigation.

2. Muscle Memory Lab (Touch Typing)
The Feature: A multi-sensory keyboard mastery module.

Inclusive Impact: Uses color-coded visual cues to build finger-to-key relationships, helping Dyspraxic students gain independence from slow handwriting.

3. Concept Mapper (Dynamic Mind Mapping)
The Feature: A non-linear "Snap-to-Grid" brainstorming canvas.

Inclusive Impact: Replaces overwhelming linear lists with a visual "dump" area for ideas, automatically organizing nodes to keep the workspace neat.

4. Atomic Goal Breakdown
The Feature: An automated task decomposer.

Inclusive Impact: Breaks large academic projects into 5-minute "Atomic" steps to prevent the ADHD "Wall of Awful" and task paralysis.

5. Focus Guardian & Proprioceptive Breaks
The Feature: A 25-minute Pomodoro timer with a mandatory 2-minute "Screen Dim" reset.

Inclusive Impact: Forces physical resets to manage muscle fatigue and sensory overload.

6. Biometric Passwordless Login
The Feature: Integrated Face & TouchID authentication.

Inclusive Impact: Eliminates the fine-motor stress of typing complex passwords, ensuring a "Zero-Friction" start to the learning session.
